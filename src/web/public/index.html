🔒 Política de Privacidad

1. Información que Recopilamos

Ghostly Guard recopila la siguiente información necesaria para el funcionamiento del bot:

  • IDs de Discord: IDs de servidores, usuarios, canales y roles para configuraciones de seguridad
  • Logs de seguridad: Registros de eventos de moderación (raids, bans, expulsiones)
  • Configuraciones: Preferencias de protección establecidas por administradores

2. Uso de la Información

La información recopilada se utiliza exclusivamente para:

  • Proporcionar protección anti-raid y anti-nuke a servidores
  • Generar logs de auditoría de seguridad
  • Mejorar las funcionalidades de detección de amenazas

3. Almacenamiento y Seguridad

Los datos se almacenan en servidores seguros con las siguientes medidas:

  • Encriptación de datos sensibles en reposo
  • Acceso restringido únicamente al propietario del bot
  • Retención limitada - los logs antiguos se eliminan automáticamente

4. Derechos de los Usuarios

Los usuarios tienen derecho a:

  • Solicitar la eliminación de sus datos
  • Acceder a los logs relacionados con su servidor
  • Desactivar la recopilación de datos eliminando el bot del servidor

📜 Términos de Servicio

1. Aceptación de Términos

Al utilizar Ghostly Guard, aceptas estos términos en su totalidad. Si no estás de acuerdo, no utilices el bot.

2. Uso Permitido

Ghostly Guard debe utilizarse únicamente para:

  • Proteger servidores de Discord contra raids y spam
  • Moderar contenido malicioso de manera automatizada
  • Gestionar configuraciones de seguridad legítimas

🚫 Prohibido: Usar el bot para evadir bans, atacar otros servidores, o cualquier actividad maliciosa.

3. Responsabilidades del Usuario

Como administrador del servidor, eres responsable de:

  • Configurar correctamente las protecciones
  • Revisar los logs de moderación
  • Usar el bot de acuerdo con los Términos de Discord
  • No abusar de las funciones de moderación automática

4. Limitación de Responsabilidad

Ghostly Guard se proporciona "tal cual". Hacemos nuestro mejor esfuerzo para mantener el bot funcionando 24/7, pero no garantizamos disponibilidad ininterrumpida.

5. Terminación

Nos reservamos el derecho de:

  • Remover el bot de servidores que violen estos términos
  • Suspender funciones en caso de uso abusivo
  • Actualizar estos términos con notificación previa
